栖息在树枝上的北长尾山雀，大庆市，中国黑龙江省

大寒（Dahan）是二十四节气中的最后一个节气。它意味着一年中最寒冷的时刻。大寒过后，立春开始，标志着新一轮节气的开始。 随着大寒的临近，过年的喜庆气氛也随之浓厚起来。大寒节气往往与年末相吻合，许多相关习俗都蕴含着辞旧迎新的寓意。

它们常在取食时发出细微而尖细的“吇吇”声（sii-sii-sii）。
